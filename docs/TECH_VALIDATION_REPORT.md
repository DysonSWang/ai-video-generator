## 一、需求概述

复制一个类似视频中的AI口播视频生成工具：
- 输入竞品链接 → 自动提取文案 → AI改写原创 → 生成数字人口播视频 → 一键发布

### 目标用户
- 个人创业者
- 中小实体商家（装修、二手车、美容院等）

### 商业模式
- 按生成量收费（一条视频X元）
- 免费试用+付费解锁

### 3.4 硅基流动 TTS

**API**: https://api.siliconflow.cn/v1
**模型**: IndexTeam/IndexTTS-2
**费用**: ¥0.3/千次

**音色创建**:
```python
import requests
import base64

# 1. 上传参考音频创建音色
with open("ref_audio.mp3", "rb") as f:
    audio_data = base64.b64encode(f.read()).decode()
    audio_url = f"data:audio/mpeg;base64,{audio_data}"

payload = {
    "model": "IndexTeam/IndexTTS-2",
    "customName": "my_voice",
    "text": "这是一段参考语音。",
    "audio": audio_url
}
response = requests.post(
    "https://api.siliconflow.cn/v1/uploads/audio/voice",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json=payload
)
voice_uri = response.json()["uri"]  # "speech:my_voice:xxx:xxx"

# 2. 使用音色合成语音
payload = {
    "model": "IndexTeam/IndexTTS-2",
    "input": "你好，这是测试语音。",
    "voice": voice_uri
}
response = requests.post(
    "https://api.siliconflow.cn/v1/audio/speech",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json=payload
)
with open("output.mp3", "wb") as f:
    f.write(response.content)
```

**测试结果**:
- 状态码: 200
- 生成音频大小: 107KB
- 音质: 可接受

**状态**: ✅ 已验证，API稳定，支持音色克隆

## 四、数据流

```
                    用户上传
                        │
                        ▼
┌─────────────────────────────────────────────────────┐
│  1. 素材输入                                      │
│     - 竞品视频链接                                 │

└─────────────────────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────┐
│  2. 音频提取 (Whisper)                            │
│     - 视频 → 音频                                  │
│     - 音频 → 文字转录                             │
└─────────────────────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────┐
│  3. 文案改写 (千问)                                │
│     - 原文 → 原创版本                              │
│     - 保持爆款结构，换皮不换骨                      │
└─────────────────────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────┐
│  4. 语音合成 (硅基流动)                            │
│     - 用户录音 → 音色克隆 (可选)                    │
│     - 改写文案 → TTS音频                           │
└─────────────────────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────┐
│  5. 唇形同步 (Wav2Lip)                             │
│     - 原始视频 + 音频 → 唇形同步视频               │

                        │
                        ▼
┌─────────────────────────────────────────────────────┐
│  6. 视频合成 (FFmpeg)                              │
│     - 唇形视频 + 字幕 + BGM + 片头片尾             │
└─────────────────────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────┐
│  7. 发布 (抖音API)                                  │
│     - 自动发布到选定平台                            │
└─────────────────────────────────────────────────────┘
```

## 五、异常处理

| 步骤 | 异常 | 处理 |
|------|------|------|
| 链接提取 | 链接无效/平台不支持 | 返回错误提示 |
| Whisper | 视频无音频 | 返回错误提示 |
| 千问改写 | API超时/内容拦截 | 重试2次，降级返回原文 |
| TTS合成 | 服务不可用 | 降级到其他TTS |
| Wav2Lip | GPU内存不足 | 减小分辨率，跳过该步骤 |
| 抖音发布 | API认证失败 | 引导用户重新授权 |
